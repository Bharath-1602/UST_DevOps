Static HTML Deployment

Deployed in AWS EC2 Linux Instance using Apache HTTPD

My First Web Deployment

Project

This project demonstrates how a static HTML page can be deployed on an AWS EC2 Linux instance using Apache HTTPD server. The web server hosts the HTML file and serves it over the internet.

⚙️ Steps Followed

1️⃣ Launch EC2 Linux instance
2️⃣ Connect using SSH
3️⃣ Install Apache HTTPD server
4️⃣ Start and enable HTTPD service
5️⃣ Place index.html inside /var/www/html
6️⃣ Access using Public IP

🌐 Technologies Used

  • AWS EC2
  • Linux
  • Apache HTTPD

Outcome

Successfully hosted a static webpage on EC2 and accessed it via browser using public IP, demonstrating basic web deployment fundamentals.
